Receptive ID Image Selector

Purpose

The purpose of this application is to aid in the visual learning and identification skills of autistic children. By using this tool, children can improve their ability to recognize and name various objects in a fun and interactive way.

How to Use

  1. Choose a category from the dropdown menu.
  2. Select the number of images you want to see (4 or 8).
  3. Click on the "Show Images" button to display the images.
  4. If you want to refresh the images while keeping the selected category, click the "Refresh Images" button.
